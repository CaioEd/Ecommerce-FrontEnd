# Ecommerce APP
Ecommerce Application Using React + Typescript.
<br>
Aplication takes data from API: https://github.com/CaioEd/Ecommerce-BackEnd

## Commands ##
All commands are in Makefile
```shell
1) make help
```
## Install Dependencies  ##
```shell
1) make install 
```

## Run Local  ##
```shell
1) make run 
```

## Run Build ##
```shell
1) make build
```

### Software ##
- React JS
- Typescript
